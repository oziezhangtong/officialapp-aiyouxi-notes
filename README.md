# officialapp-aiyouxi-notes

本仓库用于归档和发布多个独立的 HTML 页面，内容涵盖各类静态页面资源，不针对任何特定域名或网站。

## 项目简介

这是一个通用型 HTML 页面归档仓库，主要用于存放和展示独立的网页文件。每个 HTML 页面均为自包含的静态资源，可直接在浏览器中打开查看，无需额外依赖或后端服务。

## 目录说明

仓库目录结构如下：

```
/
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面引用的公共资源（如样式、脚本、图片等）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件对应一个独立的 HTML 页面归档。
- `assets/` 目录存放页面可能共享的静态资源，便于统一管理。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可离线使用或部署到任意静态托管平台。
- 页面内容可能包含文本、样式、交互功能等，但不会收集用户信息或进行数据追踪。
- 归档页面不指向任何外部链接或特定服务，仅作为静态内容展示。
